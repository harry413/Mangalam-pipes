## How to Run
### Option 1 — Open directly in a browser
Double-click index.html. Everything works locally except the Google Fonts (requires internet). The layout will fall back to system serif/sans-serif fonts if offline.
### Option 2 — Local development server (recommended)
A local server prevents any browser CORS restrictions on resource loading.
Using VS Code Live Server

### Install the Live Server extension.
Right-click index.html → Open with Live Server.
